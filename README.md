# HOTEL RESERVATION AND MANAGEMENT SYSTEM - Pristine Hotels
<img width="3456" height="2304" alt="image" src="https://github.com/user-attachments/assets/98d1043c-e1fa-4eba-b0c3-c337845b2c57" />

Organizes room reservations, guest info, and payments with backup.

## **Objectives:**
Apply advanced SQL, triggers, and data recovery methods.

## **Major Features:**

- Guests, Rooms, Reservations, Payments, Staff.
- Procedure for room availability.
- Trigger for check-in/out.
- Backup and user security.

## **Deliverables:**

1. ERD, Table creation scripts, sample data.
2. CRUD SQLs for room bookings.
3. Joins and Subqueries for availability reports.
4. Stored Procedure: Reservation confirmation.
5. Trigger: Room status update.
6. View: Occupancy summary.
7. Import/Export: Guest list.
8. Backup and restore demo.
9. Security roles.
10. Full project documentation and PowerPoint.

## **How to Run:**
**Full Documentation** [access here](https://www.notion.so/HOTEL-RESERVATION-AND-MANAGEMENT-SYSTEM-2c707eb6431d80819df5c9fa85983d82?source=copy_link)
1. Enable XAMPP or WAMP Apache and MySql to initiate the local server.
2. Pull the repository and make sure that the XAMPP can execute the website by putting it on the htdocs folder.
3. Create the database by running the FULL SCRIPT from the [Full Documentation](https://www.notion.so/HOTEL-RESERVATION-AND-MANAGEMENT-SYSTEM-2c707eb6431d80819df5c9fa85983d82?source=copy_link) on localhost/phpmyadmin.
4. Access the ``Login.php`` to access the Admin Login (Credentials are in the [Full Documentation](https://www.notion.so/HOTEL-RESERVATION-AND-MANAGEMENT-SYSTEM-2c707eb6431d80819df5c9fa85983d82?source=copy_link)).
5. To access the hotel front page, access the ``hotel.php``.
